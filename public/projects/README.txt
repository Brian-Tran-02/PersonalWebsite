Project screenshots go here.

For each project, add an image named after its slug, e.g.:
  public/projects/mfa-auth.png
  public/projects/autofill-assistant.png
  public/projects/rentify.png
  public/projects/ecommerce.png

Then, in src/lib/site.ts, set the matching project's `image` field:
  image: "/projects/mfa-auth.png"

Recommended size: 1200 x 750 (16:10). PNG or JPG both work.
Until you add one, a styled placeholder is shown in the carousel.
